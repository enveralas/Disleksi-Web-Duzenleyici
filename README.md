# Disleksi Web Düzenleyici (Dyslexia Chrome Extension)

## 📌 Proje Hakkında
Bu proje, disleksi bireylerin web sitelerini daha rahat okuyabilmesi için geliştirilmiş bir Chrome uzantısıdır.  
Web sayfalarındaki metinlerin okunabilirliğini artırmak amacıyla font boyutu, satır aralığı ve harf boşluğu gibi özellikleri düzenler.

---

## 🚀 Özellikler
- 🔤 Font boyutunu artırma
- 📏 Satır aralığını düzenleme
- 🔠 Harf boşluklarını artırma
- 🎯 Seçili metni düzenleme (sağ tık menüsü)
- 🖱️ Tüm sayfayı dönüştürme (uzantı ikonuyla)
- 🔄 Aç/Kapat (Toggle) sistemi

---

## ⚙️ Kullanılan Teknolojiler
- Chrome Extension API (Manifest V3)
- JavaScript
- HTML / CSS
- DOM Manipulation
- Event-Driven Programming

---

## 🧠 Nasıl Çalışır?
- `background.js` → Kullanıcı etkileşimlerini yönetir (context menu ve ikon tıklama)
- `content.js` → Web sayfası üzerinde stil değişikliklerini uygular
- `popup` → Kullanıcı arayüzünü sağlar
- `manifest.json` → Uzantının yapılandırmasını belirler
